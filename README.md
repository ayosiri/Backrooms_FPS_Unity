# Backrooms FPS - Prototype Verison (Unity)

First-person shooter survival game inspired by the Backrooms concept, developed in Unity.

## Features
- Enemy AI with roaming behavior
- NavMesh pathfinding system
- Player detection and chase mechanics
- First-person movement and audio feedback
- Atmospheric level design inspired by the Backrooms

## Technologies Used
- Unity Engine
- C#
- NavMesh AI system

## Controls
WASD – Move  
Mouse – Look  
Shift – Sprint
Left Click - Shoot

## Build
Playable Windows build included in the repository.
